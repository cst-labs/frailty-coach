## Implementation Notes

The generated mockups are visual references, not production assets. They include useful graphic direction, especially friendly exercise illustrations and the coach avatar in Guided Companion. To implement them in the final app:

- Recreate the UI structure, spacing, color system, typography, and interaction states in HTML/CSS/JS.
- Generate or source a consistent set of exercise illustrations for the actual app assets.
- Use real icon assets from an icon library rather than copying artifacts from the mockup image.
- Keep the app's safety and evidence language aligned with `USER_GUIDE.md`.
- Preserve mobile-first accessibility: large type, large controls, high contrast, no bottom-nav overlap, and clear focus order.
